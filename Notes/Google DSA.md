# எந்த math உண்மையில் முக்கியம்?

இதுதான் actual useful.

### தேவைப்படும் concepts:

* percentages
* ratios
* powers
* logarithm basics
* modulo
* divisibility
* prime numbers
* combinatorics basics
* probability basics

Example:

* binary search → logarithmic thinking
* hashing → modulo
* DP → counting possibilities
* graphs → combinations/path counting

* logic
* sets
* relations

* recursion thinking
* combinatorics

ஆனா thinking style முழுவதும் இதிலிருந்து வரும்.

---

# 3. Complexity Analysis Math

Algorithms optimize செய்ய இது அவசியம்.

Example:

```text
O(n)
O(log n)
O(n log n)
O(n²)
```

இதைக் புரிந்துகொள்ள:

* logarithm intuition
* growth comparison

தேவைப்படும்.

Binary search ஏன் fast?
Hashmap average O(1) எப்படி?
DP ஏன் expensive?

இவை math reasoning.
