# 2026 Google Hiring Process

Typical process:

```text
1. Resume Shortlisting
2. Recruiter Call
3. Online Assessment (sometimes)
4. Technical Coding Interviews
5. System Design (if experienced)
6. Googliness / Behavioral
7. Hiring Committee
8. Team Matching
9. Offer
```

பல candidate reports-லும் 4–6 rounds common என்று வருகிறது. ([Interview Query][4])

---

# Step-by-step process

## Round 1 — Resume Screening

Resume-ல் முக்கியம்:

* Strong projects
* Good DSA
* Internships
* Open source
* Competitive coding
* Good GitHub

IIT மட்டும் இல்லை என்று recent Google employee clarify செய்திருக்கிறார். ([The Times of India][5])

---

# Round 2 — Recruiter Call

இந்த round-ல்:

* background
* projects
* preferred language
* past experience
* interest areas

கேட்பார்கள்.

---

# Round 3 — Online Assessment (OA)

சில roles-ல் வரும்.

Usually:

* 2 coding questions
* 60–90 mins

Topics:

* Trees
* Graphs
* Arrays
* DP

---

# Round 4 — Technical Interviews

இதுதான் toughest.

Usually 3–4 rounds.

ஒவ்வொரு round:

* 45 mins
* live coding
* no IDE sometimes

அவர்கள் பார்க்கும் விஷயம்:

## Important:

* brute force சொல்லுகிறாயா?
* optimize செய்கிறாயா?
* communicate செய்கிறாயா?
* edge cases பார்க்கிறாயா?
* clean code எழுதுகிறாயா?

Google interviewer reports-ல்:
“communication while coding” மிகவும் important என்று repeatedா வருகிறது. ([The Economic Times][6])

# Hiring Committee

Google unique thing:

Interviewer alone decide பண்ண மாட்டார்.

Separate hiring committee:

* all feedback review

* leveling
