# 🧠 3.1 Computation (என்ன இது?)

👉 Computation =

```text
Input → process → output
```

---

## 🧪 Example

```cpp
int x = 10;
int y = 20;

int sum = x + y;
```

👉 இது computation ✔️

# 🔥 3.3 Expressions (VERY IMPORTANT)

👉 Expression = value produce பண்ணும்

---

## 🧪 Example

```cpp
int x = 5 + 3;
```

👉 `5 + 3` = expression

---

## 🟢 Operators

```text
+   -   *   /   %
```

---

## 🧠 Constant Expression

```cpp
int x = 5 + 3;   // fixed value
```

---

# ⚙️ 3.4 Statements

👉 Statement = action

---

## 🟢 1. Selection (Decision)

```cpp
if (x > 10)
{
    std::cout << "Big";
}
```

👉 condition check ✔️

---

## 🟢 2. Iteration (Loop)

```cpp
for (int i = 0; i < 5; i++)
{
    std::cout << i << '\n';
}
```

👉 repeat ✔️

---

# 🧠 3.5 Functions (VERY IMPORTANT 🔥)

👉 function = reusable code

---

## 🧪 Example

```cpp
int add(int a, int b)
{
    return a + b;
}
```

---

👉 use:

```cpp
int result = add(5, 3);
```

---

# 🧠 Why function?

👉 benefits:

* reuse
* clean code
* easy debugging

---

# 📦 3.6 vector (NEW 🔥)

👉 vector = dynamic array

---

## 🧪 Example

```cpp
#include <vector>

std::vector<int> v = {1,2,3};
```

---

## Loop through vector

```cpp
for (int i = 0; i < v.size(); i++)
{
    std::cout << v[i] << '\n';
}
```
