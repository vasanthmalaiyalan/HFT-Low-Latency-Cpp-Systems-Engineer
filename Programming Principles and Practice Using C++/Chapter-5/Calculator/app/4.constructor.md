# 🧠 1. Constructor இல்லாமல் எப்படி வேலை செய்கிறது?

உன் `Token` இப்படியிருக்கு என்று வைத்துக்கொள்:

```cpp
struct Token {
    char kind;
    double value;
};
```

👉 இங்கே **user-defined constructor இல்லை**.

```cpp
return Token{ch, 0};
```

👉 இது:

```text
Aggregate initialization ✔️
```

```text
kind  = ch
value = 0
```

# 🟢 2. Full Example

```cpp
#include <iostream>

struct Token {
    char kind;
    double value;
};

Token get_token(char ch) {
    return Token{ch, 0};   // aggregate initialization
}

int main() {
    Token t = get_token('+');

    std::cout << t.kind << " " << t.value << '\n';
}
```

👉 Output:

```text
+ 0
```

# 🟢 3. Important Condition

👉 இந்த method வேலை செய்ய:

```text
struct/class must be aggregate type ✔️
```

---

## ❌ Aggregate ஆக இருக்கக் கூடாத condition:

```cpp
class Token {
public:
    char kind;
    double value;

    Token(char k) {}   // constructor இருந்தால் ❌
};
```

👉 இப்போ:

```cpp
Token{ch, 0}; ❌ (aggregate இல்லை)
```

---

# 🟢 4. Constructor இருந்தால்?

```cpp
class Token {
public:
    char kind;
    double value;

    Token(char k, double v) : kind{k}, value{v} {}
};
```

👉 இப்போ:

```cpp
return Token{ch, 0};
```

👉 இது:

```text
constructor call ஆகும் ✔️
```

# 🔥 5. Summary Difference

| Situation           | Behavior                 |
| ------------------- | ------------------------ |
| No constructor      | Aggregate initialization |
| Constructor present | Constructor call         |
