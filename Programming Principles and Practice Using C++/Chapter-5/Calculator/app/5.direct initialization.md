`direct member initialization, அப்படின்னா என்ன?`

சூப்பர் கேள்வி 🔥 — “**direct member initialization**”ன்னா என்னன்னு coreஆ புரிஞ்சுக்கலாம்.

---

# 🧠 1. Simple definition

```text
direct member initialization =
object create ஆகும் நேரத்திலேயே அதன் members-க்கு value set ஆகுவது
```

👉 அதாவது:

* constructor இல்லாமலே
* compiler directா memoryல values எழுதும்

---

# 🟢 2. Example

```cpp
struct Token {
    char kind;
    double value;
};

Token t{'+', 3.14};
```

---

## 💡 என்ன நடக்குது?

```text
1. memory allocate (Token)
2. kind  = '+'
3. value = 3.14
```

👉 இது தான் **direct member initialization**

---

# 🧠 3. Memory view

```text
offset 0  → kind  = '+'
offset 8  → value = 3.14
```

👉 compiler directா write பண்ணும் ✔️

---

# 🔥 4. இது constructor இல்லாமலே நடக்கும்

```cpp
struct Token {
    char kind;
    double value;
};
```

👉 constructor இல்லை

👉 ஆனாலும்:

```cpp
Token t{'+', 3.14};
```

✔ வேலை செய்கிறது

---

# 🟢 5. இது assignment இல்ல

👉 ❌ assignment:

```cpp
Token t;
t.kind = '+';
t.value = 3.14;
```

---

👉 ✔ direct initialization:

```cpp
Token t{'+', 3.14};
```

---

# 🧠 Difference

| Type           | என்ன நடக்கும்                   |
| -------------- | ------------------------------- |
| Initialization | create + set values             |
| Assignment     | already created → change values |

---

# 🟢 6. எப்படி compiler பார்க்கும்?

👉 Conceptually:

```text
kind  ← '+'
value ← 3.14
```

👉 function call இல்லை
👉 constructor இல்லை

---

# 🔥 7. Why “direct”?

👉 “direct”ன்னா:

```text
intermediate steps இல்லாமல்
directா memory write ✔
```

---

# 🟢 8. When it works?

👉 only when:

```text
class/struct = aggregate type
(no user-defined constructor)
```

---

# ❌ When it won’t work

```cpp
class Token {
public:
    char kind;
    double value;

    Token(char k, double v) {}  // constructor
};
```

👉 இப்போ:

```cpp
Token t{'+', 3.14};
```

👉 constructor call ஆகும்

---

# 🧠 9. Real-world meaning

👉 இது:

```text
“just fill memory slots directly”
```
