# 🧠 Situation

Input:

```text
1 + 2
```

# 🟢 Step-by-step execution

```cpp
char ch;
std::cin >> ch;
```

👉 இது என்ன செய்கிறது?

```text
buffer: [1][ ][+][ ][2]

ch = '1'
buffer pointer → next position (space)
```

---

# 🟢 Next line

```cpp
if (isdigit(ch))
```

👉 `'1'` → digit ✔️

---

# 💥 Problem

👉 நாம் already `'1'` read பண்ணிட்டோம்

👉 ஆனால்:

```text
double val;
std::cin >> val;
```

👉 இது full number (like 123, 3.14) read பண்ணணும்

---

# 🟢 அதனால்தான் putback

```cpp
std::cin.putback(ch);
```

---

# 🧠 என்ன நடக்குது?

👉 buffer மீண்டும்:

```text
buffer: [1][ ][+][ ][2]
         ↑
      pointer back
```

👉 `'1'` மீண்டும் inputல வந்துடுச்சு ✔️

---

# 🟢 பிறகு

```cpp
std::cin >> val;
```

👉 இப்போ:

```text
val = 1
```

👉 இது:

```text
full number read ✔️
```

# 🧠 Simple Flow

```text
1. read '1' → ch
2. check digit ✔
3. putback '1'
4. read full number → val
```
