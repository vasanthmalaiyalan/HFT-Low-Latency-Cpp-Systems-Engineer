`isdigit அப்படின்னா என்ன அது என்ன செய்யும்`

சரியான கேள்வி 👍 — `isdigit` C/C++ல ரொம்ப basic ஆனாலும் முக்கியமான function.

---

# 🧠 `isdigit` என்றால் என்ன?

👉 `isdigit` என்பது **character digit (0–9) ஆ இருக்கிறதா?** என்று check பண்ணும் function.

---

# 📘 Syntax

```cpp
#include <cctype>

int isdigit(int ch);
```

---

# 🟢 Meaning

```text
'0' முதல் '9' வரை இருந்தால் → TRUE
இல்லைனா → FALSE
```

---

# 🧪 Example Code

```cpp
#include <iostream>
#include <cctype>

int main() {
    char ch = '5';

    if (isdigit(ch)) {
        std::cout << "Digit\n";
    } else {
        std::cout << "Not Digit\n";
    }
}
```

---

# 🧠 Output

```text
Digit
```

---

# 🟢 More Examples

```cpp
isdigit('3') → true ✔
isdigit('a') → false ❌
isdigit('+') → false ❌
isdigit(' ') → false ❌
```

---

# 🟢 Calculator contextல why use பண்ணுறோம்?

👉 Example:

```cpp
char ch;
std::cin >> ch;

if (isdigit(ch)) {
    std::cin.putback(ch);
    double val;
    std::cin >> val;
}
```

## 💡 Tamil Explanation

👉 Input:


1. `'1'` read ஆகும்

4. `123` முழுதும் read பண்ணுவோம்

# 🧠 Behind the scenes

👉 Equivalent:

```cpp
(ch >= '0' && ch <= '9')
```

---

# ⚠️ Important Note

👉 `isdigit` return type:

```text
int (0 or non-zero)
```

👉 so:

```cpp
if (isdigit(ch))  // correct
```
