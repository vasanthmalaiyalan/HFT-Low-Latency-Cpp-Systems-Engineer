# 🧠 1. Token memory layout

```cpp
struct Token {
    char kind;
    double value;
};
```

👉 typical (x86_64):

```text
offset 0  → kind (1 byte)
offset 1–7 → padding (alignment)
offset 8  → value (8 bytes)
```
