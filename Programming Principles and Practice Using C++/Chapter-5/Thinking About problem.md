# 🧠 Core Idea

👉 Programming ஆரம்பிக்கணும்னா:

> **முதலில் யோசிக்கணும் — உடனே code எழுதக்கூடாது ❌**

---

# 🟢 1. First Step: Think 🧠

👉 Book சொல்லுது:

> “Think about what the program should do”

### Tamil:

👉 Program என்ன செய்யணும்?

Example:

* calculatorனா:

  * input எப்படி இருக்கும்?
  * output எப்படி காட்டணும்?

---

## 💡 Example Thinking

👉 User input:

```text
2 + 3 * 4
```

👉 Questions:

* spaces இருக்குமா?
* decimal வருமா? (`3.1`)
* order of operations follow பண்ணணுமா?

👉 இதுதான் thinking 🔥

---

# 🟢 2. Interaction பற்றி யோசிக்க

👉 User எப்படி program use பண்ணுவான்?

### Example:

```text
Enter expression: 2+3*4
Result: 14
```

👉 இது UX (user interaction)

---

# 🟢 3. Write rough idea ✍️

👉 First idea perfect இருக்காது ❌

👉 Write something like:

```text
1. input read பண்ணு
2. numbers extract பண்ணு
3. operations apply பண்ணு
4. result print பண்ணு
```

👉 இது தான் **first draft**

---

# ⚠️ 4. Find mistakes in your idea

👉 Book சொல்ற முக்கியமான point:

> “See what’s wrong with your first idea”

### Tamil:

👉 உன் planல என்ன தவறு இருக்குனு பாரு

Example:

* operator precedence handle பண்ணல ❌
* invalid input consider பண்ணல ❌

---

# 🟢 5. Discuss with others 🗣️

👉 Book சொல்ற powerful trick:

> “Explain to a friend”

👉 Why?

* friend question கேட்பான்
* உன் logic break ஆகும் 😅
* mistakes reveal ஆகும்

👉 paper / computer:
❌ question கேட்காது

---

# 🟢 6. No universal method ❌

👉 Important truth:

> “No one method works for all problems”

👉 meaning:

* எல்லா problemக்கும் ஒரே solution approach கிடையாது

---

# 🟢 7. Strategy (for small problems)

👉 Book simple strategy குடுக்குது:

1. Think
2. Sketch idea
3. Test idea
4. Improve

---

# 🔁 8. Iterative Process (VERY IMPORTANT)

👉 Book சொல்ற biggest idea:

```text
Idea → Try → Wrong → Fix → Improve → Repeat
```

👉 Perfect solution immediately வராது ❌

---

# 🟢 9. Be Skeptical 🧐

👉 Book advice:

> “Don’t blindly trust the solution”

### Tamil:

👉 நீயே உன் code-ஐ doubt பண்ணு

---

# 🟢 10. Journey > Final Code 🔥

👉 Most important line:

> “Journey is more important than final version”

### Meaning:

👉 Final code மட்டும் important இல்லை

👉 How you reached there:

* mistakes
* thinking
* redesign

👉 அதுதான் real learning 💯

---

# 💥 Real Developer Truth

👉 Professionals:

* first attempt ❌ wrong
* second attempt ❌ still wrong
* final version ✔️ after many fixes
