# 🟢 §5.3 Back to Calculator

👉 இப்ப actual build start

### Concept: Tokens

👉 Input:

```text
2 + 3
```

👉 Split பண்ணுறது:

```text
2   +   3
```

👉 இவை தான் **tokens**

---

## 🔹 Token types:

* number → `2`, `3`
* operator → `+`, `-`, `*`, `/`

---

# 🟢 §5.4 Grammar

👉 இது next level 🔥

👉 Rules define பண்ணுறது:

```text
Expression → Term + Term
Term → Number * Number
```

👉 இதை தான் **grammar** சொல்வாங்க

## 🧠 Simple Tamil

👉 Grammar =
