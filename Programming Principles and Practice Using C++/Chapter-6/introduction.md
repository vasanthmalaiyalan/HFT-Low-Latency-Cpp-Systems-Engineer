# 🧠 Core Idea

👉 Beginner thinks:

```text id="7vf7qn"
“Program works → finished”
```

👉 Professional knows:

```text id="0jlwm8"
“Program works → real work starts now”
```

---

# 🟢 1. “Program starts running reasonably”

👉 Book says:

> “When your program first starts running reasonably, you’re probably about halfway finished.”

---

## 💡 Tamil Meaning

👉 Program:

* compile ஆகுது ✔️
* output கொடுக்குது ✔️

👉 அதனால் finishedன்னு அர்த்தம் இல்லை ❌

---

# 🧪 Example

Calculator:

```text id="0psf6m"

```

👉 if output:



👉 professional:

```text id="wtv2l6"
maintainability?
```
