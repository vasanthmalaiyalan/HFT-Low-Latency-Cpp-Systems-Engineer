# 🧩 Programs (Programன்னா என்ன?)

---

## 💡 1. Simple definition

👉 Programன்னா:

**Computerக்கு exactஆ என்ன செய்யணும் என்று சொல்லுற instructions**

---

👉 Programmingன்னா:

**அந்த instructions எழுதுறதும் + test பண்ணுறதும்**

---

## 🤖 2. Computer எப்படி?

👉 Author சொல்றார்:

👉 **Computer = மிகவும் “dumb” 😅**

---

### ❗ Meaning:

* guess பண்ணாது ❌
* common sense இல்ல ❌
* exact instruction வேண்டும் ✅

---

## 🧠 3. Human vs Computer difference

---

### 👨 Human instruction:

👉 Example:

**“Right turn எடுத்து மேல போய் leftல இருக்கும் bathroom”**

👉 Human:

* புரிஞ்சுக்குவான்
* missing steps fill பண்ணுவான்

---

### 🤖 Computer instruction:

👉 இதே சொல்லினா:

* “right என்றால் என்ன?”
* “stairs எங்கே?”
* “left எப்போ?”

👉 full confusion 😵

---

## ⚠️ 4. ஏன் precision முக்கியம்?

👉 Computerக்கு:

👉 எல்லாமே detailஆ சொல்லணும்

---

### 🔍 Example:

👉 Bathroom போக:

* chairல இருந்து எழு
* table avoid பண்ணு
* door open பண்ணு
* light on பண்ணு

👉 எல்லாமே சொல்லணும் 😬

---

## 🧱 5. Solution → Programming Language

👉 English use பண்ண முடியாது ❌
(too vague)

👉 அதனால்:

👉 **Programming language use பண்ணுறோம்**

---

## 💻 6. Programming language என்ன?

👉 Example:

* C++
* Python

---

👉 இது:

* strict grammar
* clear rules
* precise instructions

---

👉 Simpleஆ:
**“Computerக்கு புரியும் language”**

---

## 🎯 7. Important idea

👉 Computer complex வேலை செய்யும்:

👉 காரணம்:

**ஒருவர் detailed program எழுதி கொடுத்திருக்கிறார்**

👉 இப்போ:
